Projects!

I developed a program in java for Epic Seven players that optimizes the best combinations of equipment for each specific hero. For example, if you want a damage dealing (DPS) hero, then the program would iterate through the pieces of equipment to find the best combination of gear that does the most damage. Each hero in the game has different base stats, so they individually have to be coded into the software in a seperate class. I also created an algorithm to calculate gear score which will give a numerical value for the quality of a piece of gear.
